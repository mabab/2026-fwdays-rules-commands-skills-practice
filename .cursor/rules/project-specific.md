# Project-Specific Rules

## Slide Structure

- Follow structure from PRD (introduction → rules → skills → conclusions)
- Add practical examples to each topic
- Use interactive elements for engagement
- Add visualizations for complex concepts

## Workshop Content

- Focus on practical aspects
- Add real examples and cases
- Include exercises for material reinforcement
- Ensure balance between theory and practice

## Timing

- Follow total duration of 35-45 minutes
- Distribute time proportionally to topic importance
- Leave time for Q&A
- Plan buffer for unexpected questions

## Adaptability

- Ensure readability on different screens
- Test presentation on different devices
- Prepare alternative formats (PDF, SPA)
- Ensure content accessibility

## Workshop Best Practices

- Keep content engaging and interactive
- Use real-world examples
- Provide actionable takeaways
- Encourage audience participation
- Prepare for technical difficulties
